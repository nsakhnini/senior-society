Messages Help

This is the Messages page. Here you are able to send messages to your friends.

Search for a Message

This allows you to search for a specific message you sent to a friend. Search for a message by clicking in the 'Search a message' box to type or click the Microphone icon to enter text with speech using a microphone.

Access other menus or pages

Access other menus or pages by clicking the Menu Icon in the top left corner.
